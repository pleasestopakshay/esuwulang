🎯 Quick Start

Getting started with esUwUlang is as easy as saying "uwu"! Here's your first program:

owo "Hello, World! UwU"

This cute little program will output "Hello, World! UwU" to the console. The owo keyword is used for printing output!

💝 Variables

Variables in esUwUlang are declared using the uwu keyword and assigned with the 💝 operator:

uwu kawaii_number 💝 42 uwu cute_string 💝 "Hello UwU!" owo kawaii_number owo cute_string

🔢 Operators

esUwUlang uses adorable emoji operators for mathematical operations:

  • ♡ - Addition (love brings things together!)
  • 💔 - Subtraction (heartbreak takes away)
  • ✨ - Multiplication (sparkles multiply!)
  • 🌟 - Division (stars divide!)
  • 🤔 - Equality comparison
  • 😍 - Greater than
  • 🥺 - Less than

🎯 Functions

Functions are defined using the ^w^ emoticon and end with ｡◕‿◕｡:

^w^ double(x) (´･ω･`) x ✨ 2 ｡◕‿◕｡ uwu result 💝 double(21) owo result

🤔 Conditionals

Make decisions with >w< for if, T~T for else:

uwu age 💝 18 >w< age 😍 17 owo "You're an adult! owo" T~T owo "You're still young! uwu" ｡◕‿◕｡

🔄 Loops

Repeat actions with ~(˘▾˘~) while loops:

uwu i 💝 1 ~(˘▾˘~) i 🥺 6 owo "Count: " ♡ i i 💝 i ♡ 1 ｡◕‿◕｡

🚨 Error Handling

Handle errors gracefully with try-catch blocks:

(っ◔◡◔)っ uwu result 💝 10 🌟 0 (╥﹏╥) error owo "Oops! " ♡ error ｡◕‿◕｡

Complete reference for all esUwUlang keywords and operators!

owo Print/Output
>w< If statement
T~T Else statement
^w^ Function definition
(´･ω･`) Return statement
~(˘▾˘~) While loop
｡◕‿◕｡ End block
(っ◔◡◔)っ Try block
(╥﹏╥) Catch block
(╯°□°）╯ Throw error
